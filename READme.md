# Singa'Pour La Nature 🌿

Projet réalisé dans le cadre des Trophées NSI 2026 — Thème : Nature & Informatique

**Équipe :** Ingo, Swann, Jasper  
**Établissement :** International French School Singapore  

## Description
Carte interactive de Singapour avec 7 mini-jeux éducatifs sur les sites naturels emblématiques de la ville.

## Lancer le jeu
- **Version complète :** Ouvrir le dossier `Singa_Pour La Nature - Jeu complet` et lancer `main.exe`
- **Code source :** Dossier `Singa_Pour La Nature - Fichiers`, requires Python 3.11 + Pygame

## Lieux & Mini-jeux
| Lieu | Jeu |
|------|-----|
| Sungei Buloh | Jeu de canons |
| Singapore Zoo | Quiz interactif |
| Sentosa Serapong | Animation aquatique |
| Bukit Timah | Jeu d'esquive |
| Punggol Waterway | Animation naturelle |
| Jurong Bird Park | Flappy Bird |
| Botanic Gardens | Pong contre IA |
